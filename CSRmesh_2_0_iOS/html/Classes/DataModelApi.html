sendData - Send data to device. The callback didSendData is invoked upon success. The MeshServiceApi:didTimeoutMessage is invoked on failure.

- (NSNumber *)sendData:(NSNumber *)deviceId data:(NSData *)data success:(void ( ^ ) ( NSNumber *deviceId , NSData *data ))success failure:(void ( ^ ) ( NSError *error ))failure

Parameters

deviceId
  • (NSNumber *) The ID of this device. Refer to the delegate MeshServiceApi.didAssociateDevice
data
  • (NSData *) Data, up to 250 octets.
success
  • Block called upon successful execution, if nil then data will be sent by Block Data transfer otherwise Stream data transfer
failure
  • Block called upon error

Return Value

meshRequestId - (NSNumber ) The id of the request. Pair up with the id returned in didGetPowerState. For block data transfers, if the length of (NSData )data exceeds 10 then the request will be rejected and a nil returned.
