– togglePowerState:success:failure:

  • @brief togglePowerState - If the current PowerState is 0x00, Off, then the PowerState shall be set to 0x01, On. If the current PowerState is 0x01, On, then the PowerState shall be set to 0x00, Off. If the current PowerState is 0x02, Standby, then the PowerState shall be set to 0x03, OnFromStandby. If the current PowerState is 0x03, OnFromStandby, then the PowerState shall be set to 0x02, Standby.
