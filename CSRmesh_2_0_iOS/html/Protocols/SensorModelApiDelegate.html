– didGetSensorValue:sensors:meshRequestId:

didGetSensorValue. An acknowledgement to the request to set the Sensor State. Provides the value for up to 2 sensor types. The value is of variable length from 0 to 4 octets. This callback will be replaced with success and failure code blocks in the next version therefore please discontinue its use.

- (void)didGetSensorValue:(NSNumber *)deviceId sensors:(NSDictionary *)sensors meshRequestId:(NSNumber *)meshRequestId

Parameters

deviceId
  • (NSNumber *) The ID of this device. Refer to the delegate MeshServiceApi.didAssociateDevice
sensors
  • (NSDictionary) - The key is the NSNumber of the sensor Type as an unsigned short and the Value is the float of the sensor value wrapped in an NSNumber. Temperature value will be of type float in units of kelvin and in steps of 1/32 or 0.03125
meshRequestId
  • (NSNumber *) The ID assigned to the Api call that triggered this callback.
