Parameters

deviceId
  • (NSNumber *) The ID of this device. Refer to the delegate MeshServiceApi.didAssociateDevice
bearerRelayActive
  • (NSNumber *) Unsigned Short Integer of the Bitfield of active relay bearers with bits mapped as follows Bit 0 = le_advertising Bit 1 = le_gatt_server Bit 2 = udp_ip_4 Bit 3 = udp_ip_6 Bit 4 = tcp_ip Bit 5 = wifi_beacon_bearer
bearerEnabled
  • (NSNumber *) Unsigned Short Integer of the Bitfield of enabled bearers with bits mapps as follows Bit 0 = le_advertising Bit 1 = le_gatt_server Bit 2 = udp_ip_4 Bit 3 = udp_ip_6 Bit 4 = tcp_ip Bit 5 = wifi_beacon_bearer
promiscuous
  • (NSNumber *) Unsigned Short Integer of the bitfield where each bit determines if the given bearer promiscuity is enabled. Enabled in this context means that the bearer can relay all traffic it receives even if it does not match any known NetworkKeys Bit 0 = le_advertising Bit 1 = le_gatt_server
